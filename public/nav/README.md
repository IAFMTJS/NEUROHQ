# Nav-icoontjes (PNG)

De onderste navigatiebalk laadt deze PNG’s uit deze map. Gebruik **exact** deze bestandsnamen (hoofdletters en spaties tellen mee):

- `Dashboard.png` — HQ
- `Assistent.png` — Assistant
- `Missions.png` — Missions
- `Budget.png` — Budget
- `Growth.png` — Growth
- `Strategy.png` — Strategy
- `Insights.png` — Insights
- `Layer 6.png` — Design
- `Settings.png` — Settings

**Formaat:** 20×20 px of 40×40 px (wordt geschaald). Transparante achtergrond werkt het best.

Als een bestand ontbreekt of de naam wijzigt, valt die knop automatisch terug op het SVG-icoon. Wijzig dan ook `navLinks` in `components/ui/BottomNavigation.tsx`.
