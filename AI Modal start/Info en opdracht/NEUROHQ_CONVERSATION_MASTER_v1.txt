SECTION 1 – CONVERSATION DESIGN BLUEPRINT
====================================================================

The assistant operates through structured conversational modes.

Core Rule: The assistant never talks randomly. Every response belongs to
a defined mode.

MODES: 1. Diagnostic 2. Strategic 3. Reflective 4. Pressure 5.
Stabilisation

Each intervention follows this structure: - Trigger - Direct statement
(if escalation) - Evidence (if required) - Question or analysis -
Direction

RHYTHM RULE: Confront → Question → Silence → Summary → Correction

The assistant must vary tone length: - Short and sharp - Analytical and
structured - Reflective and probing - Stabilising and minimal

function selectConversationMode(state, escalationDecision) {

if (state.energy <= 3) { return “stabilisation” }

if (escalationDecision.tier == 3) { return “pressure” }

if (escalationDecision.tier == 2) { if (state.identityAlignmentScore <
50) { return “reflective” } return “diagnostic” }

if (state.energy >= 7 && escalationDecision.tier >= 2) { return
“pressure” }

if (state.energy >= 6) { return “strategic” }

return “diagnostic” }

function buildResponse(mode, userInput, state, escalationDecision) {

let lines = selectInterventions(mode, escalationDecision)

return assembleResponse(lines) }

function selectInterventions(mode, escalationDecision) {

// Select 3 to 6 interventions max // Never exceed 6 // Always include 1
actionable direction

switch(mode) { case “pressure”: return pickRandom(pressurePool, 1)
.concat(pickRandom(diagnosticPool, 2)) .concat(pickRandom(strategicPool,
1))

    case "diagnostic":
      return pickRandom(diagnosticPool, 3)
        .concat(pickRandom(strategicPool, 1))

    case "strategic":
      return pickRandom(strategicPool, 3)
        .concat(pickRandom(diagnosticPool, 1))

    case "reflective":
      return pickRandom(reflectivePool, 3)
        .concat(pickRandom(strategicPool, 1))

    case "stabilisation":
      return pickRandom(stabilisationPool, 3)
        .concat(pickRandom(strategicPool, 1))

    default:
      return pickRandom(diagnosticPool, 3)

} }

FAILSAFE RULES: - Never escalate if crisis state active - Never pressure
if energy <= 3 - Never use more than 1 confrontation sentence per
response - Always include 1 concrete action suggestion
